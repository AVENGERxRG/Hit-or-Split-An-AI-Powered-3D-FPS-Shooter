Hit or Split is an AI-powered 3D First Person Shooter (FPS) game developed using Unreal Engine

Features :

AI-driven enemy patrol, detection, and attack system

Realistic FPS weapon shooting mechanics

Multiplayer-inspired game environment

Advanced Unreal Engine Blueprint system

Smooth UI/HUD with health, ammo, and score tracking

Optimized graphics and lighting for stable performance

Interactive maps and combat gameplay

Technologies Used :

Game Engine: Unreal Engine 5

Programming: Blueprint Visual Scripting

3D Assets: Blender / Unreal Marketplace Assets

Audio: Unreal Audio System

Version Control: Git & GitHub

Gameplay Highlights :

Enemy AI with patrol and combat behaviour

Dynamic shooting and hit detection

FPS combat environment with realistic effects

Optimized lighting and environment rendering

System Requirements :

Windows 10/11 (64-bit)

Intel i5 / Ryzen 5 or higher

8 GB RAM minimum

NVIDIA GTX 1050 / AMD equivalent or above

DirectX 12 compatible GPU
